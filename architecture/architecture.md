## Software Architecture [Back](./../README.md)

- Software Architecture(軟件體系架構): 源於**建築學**, 只有可重複, 可複用的才能稱作體系結構.
- 因為軟件的體積越來越大, 因此需要體系結構來組織整個軟件的框架.
- Software Architecure(軟件體系結構) = Component(組件) + Connector(連接件) + Constrains(約束)
