## How to write menu button with CSS [Back](./qa.md)

```html
<div class="menu-btn">
    <div class="inline"></div>
</div>
```

```css
.menu-btn {
    width: 25px;
    height: 3px;
    padding: 3px 0;
    border-top: solid 3px #000;
    border-bottom: solid 3px #000;
    cursor: pointer;
}

.menu-btn .inline {
    width: 100%;
    height: 100%;
    background-color: #000;
}
```
